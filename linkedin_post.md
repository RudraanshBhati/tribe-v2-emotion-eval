# LinkedIn Post — TRIBE v2 Emotion Validation Experiment

---

I spent the past few weeks running a validation experiment I could not find anywhere else in the literature, so I built it myself.

**The question:** Can predicted brain activations help a language model understand emotion better?

---

**What TRIBE v2 is**

META's TRIBE v2 (Facebook AI Research) is a pretrained brain encoding model. You feed it text or audio, and it predicts what fMRI activation pattern a human brain would produce while processing that input — across 20,484 cortical vertices from fsaverage5 space. It is not a language model. It is a neural simulator, trained on real fMRI data from the Natural Scenes Dataset and other corpora.

The model has two encoders: a LLaMA 3.2 text encoder and a Wav2VecBert audio encoder. Which means the predicted activations change based not just on what is said, but on how it is said.

---

**The experiment**

I designed a 3-condition A/B/C test on 100 emotion-labelled text samples (happy, angry, sad, fearful) across three difficulty levels: explicit ("I feel devastated"), implicit ("it was my last day before retirement"), and sarcastic ("oh great, another Monday").

- **Condition A:** Text only — baseline, what llama3.1:8b achieves without any brain signal
- **Condition B:** Text + real TRIBE v2 neural context — the test condition
- **Condition C:** Text + shuffled (mismatched) neural context — the control

If B > A > C, the brain signal is real. If B ≈ C, the context is noise.

I ran this twice: once with plain text input to TRIBE v2, and once with emotionally expressive audio generated by ElevenLabs (12 voices, 3 per emotion, with per-emotion voice settings).

---

**What I found**

The signal is partial — and the honest reporting of that is itself useful.

**LLM evaluation results (300 Ollama calls, temperature 0):**

- Overall: A=52%, B=55%, C=54% — small but consistent B > A
- Explicit emotions: B=67.5% vs A=60% vs C=60% — clean B > A = C pattern
- Implicit emotions: B=50% vs C=55% — shuffled context outperforms real (noise pattern)
- Sarcastic: flat at 40% across all conditions — expected

**Emotional audio made the explicit result stronger** (+5pp over text-only input to TRIBE v2). ElevenLabs prosody reaches the Wav2VecBert encoder and shifts the predicted activation patterns.

But here is where it gets more interesting.

**I also ran a direct ROI classifier** — logistic regression and SVM on the 7 ROI mean activations, no LLM involved:

- Text-based activations: SVM 26% (barely above 25% chance)
- Emotional audio activations: SVM 33%, RF 32% — clearly above chance
- Per-category (LogReg, explicit): 47.5% from raw brain data alone
- Per-category (LogReg, implicit, emotional audio): 47.5% — nearly double chance

So the raw activations *do* encode emotion above chance. The signal that the LLM formatter is not surfacing is still there in the data. The bottleneck is the natural language translation layer, not the brain encoding model.

**Most discriminative ROIs:** ACC (anterior cingulate cortex — conflict/uncertainty) and amygdala (threat/arousal). Consistent with the emotion neuroscience literature.

---

**What this means**

TRIBE v2 is doing something real. Even with index-based (not anatomically validated) ROI slices and only 100 samples, the activations discriminate emotion above chance when you bypass the LLM formatter entirely.

The gap between the raw classifier result (47.5% explicit, 47.5% implicit) and the LLM eval result (67.5% explicit, 50% implicit) tells you exactly where the next work lives: in a better formatter, anatomical parcellation (Glasser HCP-MMP1.0 has 360 real regions), or a retrieval-augmented approach that feeds the activations directly rather than translating them to text.

---

**Limitations (honest)**

- 100 samples is directional, not definitive
- ROI slices are index-based approximations, not real atlas regions
- Free-tier ElevenLabs (10K chars/month) limits reproducibility without cost
- llama3.1:8b is not a specialist emotion model
